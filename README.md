# SketchPad Studio

SketchPad Studio is a modern browser-based drawing web app built with React, TypeScript, Tailwind CSS, and the HTML Canvas API. It lets users draw, erase, create simple shapes, undo and redo changes, auto-save their work locally, and export their drawing as a PNG image.

This project was built as a portfolio piece to practise frontend development, canvas interaction, responsive UI design, and component-based React architecture.
